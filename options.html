Tab Session Saver — Settings

Zap

How aggressively the ⚡ button trims tabs in the current window. Closed tabs can be restored with Ctrl+Shift+T.

Backup & Restore

Export

Download all your saved sessions as a JSON file. Keep it somewhere safe as a backup.

Import

Load sessions from a JSON file. Imported sessions are added to what you already have — nothing is overwritten. Sessions that duplicate an existing one (same name and same tabs) are skipped.
